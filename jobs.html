Website Content Coordinator

Reference Number: SEA12

Salary: $78,000 - $80,000
Reporting Line: Communication Manager

Job Description

The Website Content Coordinator is responsible for updating and maintaining website content, ensuring information is accurate, up-to-date, and easy for users to understand.

Key Responsibilities

  • Maintain and update website content using HTML and CSS.
  • Ensure HTML and CSS lines meet standard validation.
  • Ensure Web Accessibility Evaluation is met.
  • Delpoy Website in GitHub.

Requirements

Essential:

  1. Experience in HTML and CSS.
  2. Strong written and communication skills.
  3. Experience in using GitHub such as performing commits, pushes, and merges.

Preferable:

  • Experience in the sustainability industry.
  • Knowledge of web accessibility principles.
  • Adaptability and Organization Skills.
